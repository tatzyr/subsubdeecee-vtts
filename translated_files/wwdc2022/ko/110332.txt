나마스카! 안녕하세요, WWDC22에 오신 것을 환영합니다.

제 이름은 Create ML 팀의 엔지니어인 Vrushali Mundhe이며, Create ML의 새로운 기능을 공유하게 되어 기쁩니다.

Create ML을 사용하면 수집한 데이터로 강력한 기계 학습 모델을 쉽게 훈련시켜 독특한 경험을 제공하고 앱을 향상시킬 수 있습니다.

Xcode와 함께 번들로 제공되는 앱으로 ML 배송을 만들어 코드 없이 Mac에서 바로 Core ML 모델을 빠르게 빌드하고 훈련할 수 있습니다.

Create ML은 SDK에서 Swift 프레임워크로 배송할 수도 있습니다.

API를 사용하면 모델 생성을 쉽게 자동화하거나 자신의 앱 내에서 직접 교육으로 구동되는 역동적인 경험을 만들 수 있습니다.

Create ML의 핵심 기능에 대해 자세히 알아보려면, 이 이전 세션을 확인할 수 있습니다.

이 세션에서, 우리는 Create ML의 새로운 기능에 대해 이야기할 것이다.

우리는 당신의 모델의 정확성과 유용성을 평가하는 데 도움이 될 수 있는 Create ML 앱의 새로운 기능으로 시작할 것입니다.

그런 다음 Create ML 프레임워크, 확장된 기능 및 이제 자신의 앱에 대한 모델을 고도로 사용자 정의할 수 있는 능력에 관심을 돌릴 것입니다.

모델 생성을 위한 일반적인 워크플로우를 검토하는 것으로 시작합시다.

확인된 작업을 감안할 때, 당신은 데이터를 수집하고 주석을 뗷는 것으로 시작합니다.

예를 들어, 식료품을 시각적으로 식별하고 싶습니다.

그런 다음 Create ML은 앱에서 사용할 수 있는 이 데이터에서 모델을 훈련시키는 데 도움이 될 것입니다.

그러나, 이 모델을 사용하기 전에, 중요한 단계는 그것이 얼마나 잘 작동하는지 평가하는 것입니다; 여기서, 훈련 세트의 일부가 아닌 이미지에서 모델이 얼마나 정확한지 보는 것입니다.

평가에 따라, 추가 데이터와 수정된 교육 설정으로 모델을 반복하거나, 모델이 충분히 잘 수행되면 앱에 통합할 준비가 되면, 이 평가 단계에 더 집중하고 싶습니다.

평가를 수행할 때, 우리는 종종 훈련에서 나온 새로운 데이터에 대한 모델을 테스트하여 측정된 일련의 메트릭으로 전환합니다.

최상위 정확도 메트릭을 보거나 클래스별 통계에 뛰어들어 모델의 행동에 대한 일반적인 감각과 훈련된 것 이상으로 일반화할 수 있는 능력을 얻을 수 있습니다.

궁극적으로, 모델은 앱에서 데이터 기반 경험을 강화할 책임이 있으며, 평가 중에 입력 또는 시나리오 범주 측면에서 모델의 주요 강점과 약점을 식별하고 싶습니다. 잘 작동하거나 기대에 미치지 못할 수 있습니다.

Create ML 앱에는 모델 생성 여정의 이 부분에 도움이 될 수 있는 새로운 기능이 있습니다.

이 앱은 탐험을 시작하기에 좋은 장소가 될 수 있는 가장 중요한 평가 세부 사항 중 일부를 선택했습니다.

이 작은 썸네일에서 말하기는 어렵지만, 더 나은 보기 위해 이 이미지를 클릭하여 확인해 봅시다.

그것은 또한 이 왼쪽과 오른쪽 화살표를 사용하여 예시를 탐색할 수 있게 해준다.

라이브 미리보기는 즉각적인 피드백을 가능하게 한다.

이미지 분류기, 핸드 액션 분류기 및 바디 액션 분류기 템플릿으로 확장되었습니다.

또한 연결된 웹캠에서 선택할 수 있도록 기능을 확장했으며 macOS Ventura에서 연속성 카메라도 지원합니다.

그것은 Create ML 앱의 새로운 기능에 대한 간략한 요약입니다.

Create ML 프레임워크의 새로운 기능에 대해 논의하기 위해 전환합시다.

Create ML 프레임워크는 macOS, iOS 및 iPadOS에서 사용할 수 있습니다.

작업 지원은 플랫폼마다 다르다는 점에 유의하십시오.

예를 들어, 테이블 분류기와 회귀자는 어디에서나 사용할 수 있지만, 비디오와 관련된 것과 같이 더 큰 데이터와 계산 요구 사항이 있는 작업 중 일부는 macOS가 필요합니다.

당신이 가질 수 있는 한 가지 일반적인 질문은 "내 아이디어를 Create ML의 미리 정의된 작업 중 하나에 매핑할 수 없다면 어떨까요?"입니다.

이 질문에 답하기 위해, 우리는 Create ML 제품군에 새로운 구성원을 소개합니다: Create ML Components.

Create ML Components는 Create ML의 기본 빌딩 블록을 노출합니다.

그것들을 함께 결합하여 사용 사례에 맞춤화된 파이프라인과 모델을 만들 수 있습니다.
