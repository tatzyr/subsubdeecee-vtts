이것은 사파리의 브라우저 개발 도구인 웹 인스펙터의 모든 기능에 대한 5분짜리 연습입니다.

시작하자.

웹 인스펙터를 보려면, 먼저 개발 메뉴를 활성화해야 합니다.

개발 메뉴를 활성화하려면, 사파리 메뉴에서 기본 설정을 클릭하세요.

고급 섹션으로 이동하여 섹션 하단의 확인란을 선택하여 개발 옵션을 활성화하십시오.

이러한 옵션을 활성화한 후, "Command + Option + I"를 눌러 웹 인스펙터를 열거나, 개발 메뉴에서 웹 인스펙터 보기 옵션을 선택하십시오.

웹 인스펙터의 도구는 탭으로 나뉩니다.

탭을 닫고 더하기 탭으로 다시 추가할 수 있습니다.

모든 탭의 하단에는 콘솔 바가 있다.

이 막대를 미니 콘솔 창으로 확장하려면 이스케이프를 클릭하세요.

측면에는 내비게이션과 세부 사항을 위한 두 개의 사이드바가 있다.

내비게이션 바는 창 하단에 자체 필터가 있습니다.

내비게이션 사이드바에서 무언가를 선택하면 주요 콘텐츠 영역과 세부 정보 사이드바의 내용이 변경됩니다.

각 사이드바는 표시하거나 숨길 수 있습니다.

그리고 주요 콘텐츠 영역이 있습니다.

이 영역은 내비게이션 기록을 저장하므로 앞뒤로 이동할 수 있습니다.

현재 자원의 전체 경로를 보여주는 선택 경로 표시줄도 있습니다.

탭 상단에는 도킹 옵션, 새로 고침 버튼, 다운로드 버튼, 페이지 통계 대시보드, 모든 리소스를 검색할 수 있는 글로벌 검색 표시줄 및 요소 선택 버튼이 있는 도구 모음이 있습니다.

요소를 선택하면, 첫 번째 탭인 요소 탭으로 이동합니다.

요소 탭의 주요 내용은 편집 가능한 DOM 트리 개요입니다.

세부 사항 사이드바에는 세 개의 섹션이 있습니다.

노드 섹션은 선택한 노드와 관련된 추가 정보를 보여줍니다.

스타일 섹션에는 세 가지 견해가 있습니다: 계산, 규칙 및 시각적.

스타일을 사용하면 CSS 속성을 보고 편집할 수 있습니다.

레이어 섹션은 선택한 요소와 관련된 합성 레이어에 대한 정보를 보여줍니다.

점프 바를 사용하여 DOM을 탐색하고 편집할 수 있습니다.

상단에는 인쇄 스타일 시트를 활성화하고, 그림자 DOM 노드를 표시하고, 합성 테두리를 표시하고, 페인트 깜박임을 활성화하는 일련의 버튼이 있습니다.

다음은 네트워크 탭입니다.

이 탭은 현재 웹 페이지에서 로드된 모든 리소스를 보여줍니다.

기본 탭 기능 외에도 캐시를 비활성화하고 네트워크 항목을 지우는 버튼이 있습니다.

헤더를 보려면, 세부 정보 사이드바를 확인하세요.

선택한 리소스를 보려면, 목록 항목 옆에 있는 점프 화살표를 클릭하세요.

다음은 리소스 탭입니다.

네트워크 탭과 마찬가지로 이 탭은 리소스 목록을 표시하고 기본 탭 기능이 있습니다.

게다가, 그것은 당신의 코드를 검사할 수 있는 세 개의 버튼이 있습니다.

예쁜 인쇄 도구는 당신의 코드를 압축 해제합니다.

유형 프로파일링은 색상 코드 유형으로 코드의 개체에 주석을 달 수 있습니다.

코드 커버리지는 실행되지 않은 코드를 어둡니다.

타임라인 탭은 웹 페이지의 성능을 확인하는 데 도움이 됩니다.

악기를 선택하고 기록 타임라인 버튼으로 다시 로드 작업과 같은 이벤트를 기록하세요.

녹화를 중단하면, 이벤트나 캡처된 프레임으로 타임라인 데이터를 검사할 수 있습니다.

자세한 내용은 녹음한 도구 중 하나를 클릭하십시오: 네트워크 요청, 레이아웃 및 렌더링, 자바스크립트 및 이벤트, 메모리 또는 자바스크립트 할당.

타임라인을 지우려면, 타임라인 지우기 버튼을 클릭하세요.

디버거에는 리소스 탭에 동일한 도구가 많이 포함되어 있습니다.

또한 왼쪽 사이드바에는 브레이크포인트 섹션이 있고 오른쪽 사이드바에는 스코프 체인이 있습니다.

중단점을 비활성화하고 일시 중지하고, 계속하고, 코드를 단계별로 진행하는 디버거 컨트롤은 탭의 왼쪽 상단 모서리에 있습니다.

저장소 탭은 검사 중인 웹 콘텐츠에 사용할 수 있는 영구 상태를 보여주며, 새로 고칠 수 있습니다.

기본적으로 쿠키, 로컬 스토리지 및 세션 스토리지를 표시합니다.

하지만 사용 가능한 경우, 이 탭은 WebSQL, IndexedDB 및 Application Cache와 같은 추가 스토리지 API도 보여줍니다.

마지막은 콘솔이야.

웹 콘텐츠로 기록된 메시지를 보여주는 것 외에도, 이 탭에는 버튼이 있습니다.

콘솔 로그를 필터링하려면, 콘솔 로그를 지우고 가비지 수집을 트리거하세요.

로그 지속성과 같은 추가 옵션을 보려면, 탭 행 끝에 있는 설정 탭으로 이동하십시오.

요약하자면, 사파리에서 개발 메뉴를 활성화하여 웹 인스펙터에 액세스할 수 있습니다.

웹 인스펙터를 사용하여 사파리에서 웹 페이지를 검사하세요.

최신 버전의 웹 인스펙터를 시도하려면, 애플 개발자 사이트에서 사파리 기술 미리보기를 다운로드하세요.

버그를 찾으면, bugreport.apple.com에서 보고하세요.

웹 인스펙터에 대한 자세한 정보와 추가 리소스를 보려면, 이 비디오와 관련된 리소스 섹션의 링크를 사용하세요.

그리고 그것은 웹 인스펙터의 빠른 연습입니다.

봐줘서 고마워.
