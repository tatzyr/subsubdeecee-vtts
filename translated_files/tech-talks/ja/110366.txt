Game Centerの概要から始めて、Game Centerをゲームに追加するのがいかに簡単かを共有し、Game Centerの主要な機能のいくつかと、それらがプレイヤーを引き付け、引き付けるのにどのように役立つかを共有し、ゲームを次のレベルに引き上げるために使用できるいくつかの追加機能と技術を共有します。Game Centerの概要から始めましょう。

Game Centerは、プレイヤーが友達とゲームの楽しさを体験できるように設計されたAppleのソーシャルゲームネットワークです。

Game Centerを使用すると、プレイヤーは友達がプレイしているゲームを見ることができ、Appleデバイス間で簡単にプレイできます。

Game Centerでは、すべてのプレイヤーがニックネームとアバターでカスタマイズできるプロフィールを持っています。

プロフィールから、プレイヤーはどの成果をアンロックしたか、最近プレイした友人やゲームを見ることができ、友人はゲームを楽しむ上で非常に重要な部分であるため、プレイヤーが簡単に接続して一緒にプレイできるようにしました。

プレイヤーは友達の提案を見たり、友達を追加したりして、ゲームにすばやく招待してスコアを比較したり、プレイしているゲームを見たり、アクティビティゲームセンターでは、友人が新しいゲームをプレイし始めると、成果のロックを解除したり、リーダーボードにランクアップしたりするなど、プレイヤーとその友人が行っているすべての楽しいことをまとめることができます。

プレイヤーがすべてのGame Centerアクティビティを見るのに最適な場所の1つは、Game Centerダッシュボードです。ここでは、プレイヤーは自分のアクティビティと友達のアクティビティを見ることができるので、最新の情報に追いつき、友達がプレイしている新しいゲームを発見することができます。

彼らはこれらのアクティビティのいずれかをタップして、友達がプレイしているゲームに簡単に飛び込むことができます。プレイヤーはゲーム内のアクセスポイントなど、いくつかの異なる方法でダッシュボードにアクセスできますが、プレイヤーが友達のアクティビティを見ることができるのはダッシュボードだけではありません。

プレイヤーがアクティビティを表示できるもう1つの方法は、Game Centerウィジェットを使用すると、プレイヤーはGame Centerウィジェットをデバイスのホーム画面に追加して、友達がプレイしているものを一目で確認できます。

これにより、プレイヤーは追加のアプリやゲームを開くことなく、友人のアクティビティを簡単に把握でき、ダッシュボードのように、任意のアクティビティをタップしてダウンロードしたり、そのゲームを開始したりできます。

Game Centerはまた、友人がゲームで挑戦したり、リーダーボードでスコアに合格したりした場合など、友人のアクティビティに関する通知をプレイヤーに送信します。しばらくプレイしていないゲームであっても、プレイヤーに通知されるため、戻ってより高いスコアを獲得する機会を逃すことはありません。

そして最後に、このアクティビティはすべて、Game CenterやFriends Are Playingのセクションや製品ページなど、アプリストア全体に表示されます。

これにより、プレイヤーはダウンロードする新しいゲームを探している瞬間に友達が何をプレイしているかを見ることができます。

ゲームにゲームセンターを追加すると、ゲームはこれらの場所のすべてに表示され、プレイヤーがゲームを発見してダウンロードするより多くの方法を提供するので、ゲームにゲームセンターを追加するにはどうすればよいですか?

App Store ConnectでGame Centerを有効にするには、[マイアプリ]セクションに移動し、ゲームを選択し、Game Centerを追加するアプリのバージョンもサイドバーで選択されていることを確認します。完了したら、ページを下にスクロールして[Game Center]チェックボックスを選択します。

次に、XcodeプロジェクトにGame Center機能を追加します。これを行うには、Xcodeに移動し、Game Center機能をダブルクリックするか、プロジェクトにドラッグします。

次に、数行のコードを追加して、プレイヤーをGame Centerにサインインし、ゲームを起動するときにGame Center機能を使用できるようにするだけです。

Game Centerを有効にすると、プレイヤーはゲームの起動時に自動的にGame Centerにサインインし、ゲームで利用可能なGame Center機能を楽しむことができることを知らせるウェルカムバナーも表示されます。

また、ゲームセンターを有効にするだけで示したように、あなたのゲームは私たちが言及したすべての場所に自動的に表示され、あなたのゲームの発見を促進し、プレイヤーとその友人を楽しませるのに役立ちます。

Game Centerの開始方法と、それがあなたとあなたのプレイヤーにもたらす価値を確認したので、Game Centerが提供するいくつかの主要な機能と、それらを使用してゲームのさらなる発見とエンゲージメントを促進する方法を見てみましょう。

このセクションでは、アイデンティティ、実績、リーダーボード、マルチプレイヤー機能のGame Centerの主な機能を使用して、プレイヤーのエンゲージメントを深め、発見をさらに高める方法を共有します。Game Centerのアイデンティティがどのように機能し、それを使って何ができるかを簡単に見てみましょう。

Game Centerを共有したように、プレイヤーはプレイしているゲーム間で一貫したアイデンティティを構築できます。

このアイデンティティはプロフィールから始まり、ゲーム活動を友達と共有することができます。

ゲームにゲームセンターを追加するだけで、プレイヤーのためにこの機能のロックを解除し、ゲームのより大きな発見が可能になります。

また、いくつかの異なる方法で、このゲームセンターのアイデンティティの上に、よりパーソナライズされた体験を構築するオプションもあります。

たとえば、プレイヤーのGame Centerアバターを表示するアクセスポイントをゲームに追加できます。プレイヤーはアクセスポイントをタップして、リーダーボードやアチーブメントの表示など、ゲームがサポートする他のゲームセンター機能に簡単にアクセスできます。

アクセスポイントが最初に表示されると、プレイヤーがロック解除したアチーブメントの数やリーダーボードのランキングなどのハイライトを表示するオプションがあり、プレイヤーは進捗状況をすばやく把握できます。

ハイライトを表示した後、アクセスポイントが折りたたんでアバターを表示し、Game Centerを実装することで、プレイヤーのニックネームとアバターにアクセスできるため、これらをゲームに直接統合できます。

たとえば、マルチプレイヤーの試合中にゲーム内にプレイヤーのアバターを表示したり、プレイヤーのニックネームとアバターをデフォルトとして使用して、オンボーディング体験をスピードアップすることもできます。

Game CenterのIDでは、プレイヤーのゲーム状態を保存できるため、進捗状況は新旧を問わずすべてのデバイスに同期され、中断したところからすぐに再開できます。

次に、成果について話しましょう。成果は、ゲームで特定の目標に到達したときにプレイヤーに知らせる特別なマイルストーンであり、プレイヤーはさまざまな成果に向けた進捗状況を追跡したり、進捗状況を友人と共有したりできます。

成果は、プレイヤーが進み続け、より深くゲームに関与するように動機づけるのに最適であり、それらをすべて収集して完了したいコンプリートに最適です。

ゲームで3種類のアチーブメントを作成でき、それぞれがプレイヤーと異なって表示されます。標準的なアチーブメントでは、プレイヤーは特定のレベルを完了するなどの概説された基準を満たす必要があり、これらはプレイヤーがその基準を満たすまでロックされているように見えます。

進歩的な成果は、プレイヤーに徐々に進歩し、成果の完了とロック解除にどれだけ近づいているかを示しています。この種の成果は、プレイヤーの進捗状況を簡単に見ることができるため、プレイヤーに進み続けるように動機づけるのに特に役立ちます。

進歩的な成果の例は、一定数の敵を倒したり、一定量の貴重な鉱物を集めたりすることです。

そして最後に、隠された成果は、プレイヤーがその目標に到達してロックを解除した瞬間まで詳細を隠します。

隠された成果は、プレイヤーがあなたのゲームでそれを発見するまで、プレイヤーのために保存したい驚きや喜びの瞬間など、ストーリーやゲームプレイのネタバレを防ぐのに役立ちます。

プレイヤーは、自分のプロフィールで完了した成果の数を見て、ゲームで達成したことの感覚を得ることができ、タップして成果の詳細を表示して、進行中の成果とまだロックされている成果を確認できます。

これは、プレイヤーが自分の進捗状況を追跡し、ゲームに戻ることを奨励し、プログレッシブアチーブアチーブメントを作成した場合、プレイヤーはロック解除に向けた進捗状況を見ることができ、アチーブメントの完了に近づいたときに知らせて、余分なモチベーションを与えます。

プレイヤーがアチーブメントを獲得すると、ゲーム内に通知を表示することでお祝いの瞬間を与えることができます。ゲームコールゲームキットでアチーブメント完了バナーを表示することでこれを行うことができます。

プレイヤーは、ダッシュボード、ゲームセンターウィジェット、アプリストアなど、デバイス全体にアチーブメントアクティビティが表示されるので、プレイヤーがゲームで新しいアチーブメントのロックを解除するたびに、友人はこのアクティビティを見て、あなたのゲームをプレイするインスピレーションを得て、自分のアチーブメントのロックを解除します。

ゲームにアチーブメントを追加するには、App Store Connectに移動してゲームとバージョンを選択し、サービスタブを選択し、アチーブメントセクションまでスクロールし、プラスアイコンをクリックしてアチーブメントを追加します。

ここでは、アチーブメントを設定し、アチーブメントの名前と説明、ポイント値、非表示かどうか、アチーブメントを複数回完了できるかどうかなど、必要な情報を入力できます。

そして、その成果は、プレイヤーが自分の進歩を追跡し、自分の成果を友人と共有するための素晴らしい方法です。

次に、リーダーボードを見てみましょう。リーダーボードのプレイヤーはゲーム内の競技に参加し、他のプレイヤーにスコアを上回るように挑戦することができます。

ゲームとのより深いエンゲージメントを奨励し、さまざまなレベルやモードのリーダーボードをゲームに設定できます。

たとえば、私たちのゲーム「コースト」では、ゲーム内のマップレベルごとに1つずつ、6つのリーダーボードを作成しました。

リーダーボードは、プレイヤーがゲームプレイ中に獲得したポイントの数と、友人の間でランク付け方法を示し、プレイヤーは友人がリーダーボードでそれらを渡すときに見ることができます。また、友人のスコアに勝つために必要なポイントの数は、ゲームに戻って合格するように動機づけます。

示したように、プレイヤーはリーダーボードのアクティビティに関する通知を受け取り、ランクを取り戻し、ランクを取り戻すモチベーションを高め、リーダーボードのグローバルビューをタップすることで、世界中の他のプレイヤーの間でどのようにランク付けされているかを確認できます。

これらの体験を有効にするには、ゲームキットを使用してゲームからスコアデータを送信するだけです。

これを行うと、Game CenterはこれらのスコアをGame Centerインターフェイスに保存して表示するか、Game Kitを使用してデータを送信および保存し、独自のカスタムインターフェイス内にリーダーボードを表示できます。

スコアデータを送信すると、プレーヤーは、デバイスに追加したGame CenterウィジェットのダッシュボードやApp Storeなど、リーダーボードのアクティビティがデバイス全体に表示されます。

これは、プレイヤーが挑戦を感じ、自分のベストスコアを打ち負かしたり、リーダーボードを登り、友人のスコアを打ち負かしたりするかどうかにかかわらず、あなたのゲームにもっと深く関与することを奨励することができます。通知では、リーダーボードは、アクティブなプレイヤーのエンゲージメントを維持するだけでなく、最近プレイしていないかもしれない人々を呼び戻して、戻ってランクを取り戻すための素晴らしい方法です。

ゲームにリーダーボードを追加するには、App Store Connectに移動してゲームの反転を選択し、[サービス]タブをクリックし、リーダーボードセクションまでスクロールし、ボタンをクリックしてリーダーボードを追加します。

クラシックまたはリミュートで作成するリーダーボードの種類を選択し、リーダーボードの名前、スコア形式、期間などの必要な情報を追加します。これは、プレイヤーが自分自身や友人に挑戦するためのエキサイティングな方法です。

次に、マルチプレイヤーに飛び込みましょう。

Game Centerは、プレイヤーが友人と簡単かつ直感的に接続、招待、プレイできるように、マルチプレイヤー機能のスイートをサポートしています。

Game Centerはリアルタイム機能とターン制の両方の機能をサポートしているため、プレイヤーはゲームを離れることなく友達と簡単に接続できます。

Game Centerを使用すると、プレイヤーはタップするだけで友達をマルチプレイヤーの試合に招待できます。Game Centerのマルチプレイヤーでは、プレイヤーはGame Centerの友達、iMessageグループ、近くのプレイヤーを招待できます。

プレイヤーが連絡先から簡単に友達を招待できるようにするために使用できるデフォルトのUIを提供し、自動マッチングにより、Appleはプレイヤーを他の競合他社と接続して、すぐにプレイを開始できるようにします。

Game Center UIを使用するか、独自のカスタムインターフェイスを使用して、この体験を提供できます。SharePlayの統合により、プレイヤーはグループFaceTime通話またはiMessageグループからマルチプレイヤーゲームを開始できます。参加者は試合にシームレスに追加され、シェアプレイは、より多くのプレイヤーとその友人をゲームに紹介するのに最適な方法です。

アプリ内イベントは、ゲームとの追加のエンゲージメントを促進するのに役立つ素晴らしい方法です。アプリ内イベントを使用して、コンペティションチャレンジ、特別なコンテンツの発売など、ゲーム内のタイムリーなイベントを宣伝できます。

人々はiOSとiPad OSのアプリストアでこれらのイベントを発見することができ、あなたのイベントを紹介し、そのリーチを拡大する新しい方法を提供します。

イベントカードには、画像やビデオ、イベント名、簡単な説明が含まれており、人々はタップしてイベントの詳細ページを開き、より多くの情報を表示したり、イベントを友人と共有したり、イベント開始時に通知を受け取ることをオプトインして、見逃すことはありません。

アプリ内イベントは、製品ページの検索や推奨事項、今日のタブなどの編集キュレーションなど、アプリストア全体に表示されます。

アプリ内イベントを提供することは、新しいプレイヤーにリーチし、現在のプレイヤーのエンゲージメントを維持し、失効したプレイヤーと再接続するのに役立ちます。

アプリ内イベントは、App Store Connectで簡単に設定できます。App Store Connectでは、一度に最大10件の承認されたイベントと、アプリストアで最大5つのイベントを公開できます。もちろん、App Store ConnectのApp Analyticsを使用して、公開されたアプリ内イベントのパフォーマンスを理解できます。

効果的なイベントメタデータを作成できるイベントの種類など、アプリ内イベントの詳細については、Apple Developer Webサイトのアプリ内イベントページをチェックし、ゲームを起動する前に、興奮を高めるための素晴らしい方法は予約注文です。

ゲームを予約注文可能にすると、人々はあなたの製品ページを表示し、ダウンロード用にリリースされる前にゲームを注文することができます。

製品ページには、ゲームに対する追加の期待を高めることができるリリース予定日も表示されます。製品ページに加えて、検索結果や今日またはゲームタブからゲームを予約することもできます。

ゲームが紹介されている場合は、App Store Connectでレビューのためにゲームとメタデータを提出すると、すべてのAppleプラットフォームで新しいゲームを予約注文できます。

予約注文のためにゲームを提供するときは、2〜180日後にダウンロードリリース日を選択します。

発売前に、マーケティング資料のアプリストアの予約注文バッジを使用して、ゲームが予約注文可能であることを顧客に知らせることができ、発売日に、ゲームは24時間以内に顧客のデバイスに自動的にダウンロードされ、自動ダウンロードを有効にした他のデバイスに自動的にダウンロードされます。

また、あなたのゲームが利用可能になったことを知らせる通知をお客様に送信し、最後に、最高のゲームを構築し、Appleデバイス間でプレイヤーに素晴らしい体験を提供するのに役立つゲーム技術の完全なスイートを提供します。

ゲームコントローラーフレームワークを使用すると、プレイヤーは物理または仮想ゲームコントローラーを使用してゲームと対話することができます。これには、サードパーティのコントローラーだけでなく、マウスキーボードとSiriのリモートメイキングを含めることができ、さらに魅力的で没入感のあるゲームプレイ体験を実現します。

Xcodeでコントローラーのサポートを有効にすると、この情報が製品ページに表示され、コントローラーを使用してゲームをプレイできることを人々に知らせます。
